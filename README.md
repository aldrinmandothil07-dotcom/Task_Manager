# Simple Task Manager

## Overview
The Simple Task Manager is a web-based application developed using HTML, CSS, and JavaScript. It allows users to manage their daily tasks efficiently by adding, completing, and deleting tasks.

## Features

### Add Task
- Enter a task name in the input field.
- Click the **Add Task** button to add a task.
- Press the **Enter** key to add a task quickly.

### View Tasks
- Displays all added tasks in a list format.

### Complete Task
- Mark tasks as completed by clicking the **Complete** button.
- Completed tasks are shown with a strikethrough effect.

### Delete Task
- Remove tasks from the list by clicking the **Delete** button.

## Technologies Used
- **HTML5** – Structure of the application
- **CSS3** – Styling and layout
- **JavaScript (ES6)** – Functionality and DOM manipulation

## How to Run the Project

1. Download or clone the project files.
2. Ensure all files are in the same folder:
   - `index.html`
   - `style.css`
   - `script.js`
3. Open `index.html` in any modern web browser.
